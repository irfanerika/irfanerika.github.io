WEB DEV

Jumat, 2 September 2016

  1. Website apa yang paling ingin kamu buat? tentang apa?

    2. Sebernanya saya juga masih bingung website apa yang kelak ingin saya buat. Namun, bila di pikir-pikir mungkin tidak jauh dari website sebuah komunitas.

    Bila ditanya kembali, komunitas apa yang ingin kamu buat? mungkin jawaban saya sekarang adalah mengenai komunitas Smartphone. Mungkin lebih spesifiknya mengenai tweak dan modding software smartphone. Baik mengenai ROM ataupun kernelnya.

    Mengapa saya sangat tertarik mengenai hal ini, mungkin jawabanya karena kita memang tidak bisa jauh dari smartphone. Kemana, dimana dan kapanpun kita selalu membawa barang yang satu ini.

  2. Hasil atau informasi apa yang paling ingin kamu sampaikan?

    3. Kelak dalam website ini saya ingin lebih berbagi kepada komunitas-komunitas smartphone yang ada di Indonesia. Hal yang ingin dibagikan mencakup berbagai hal baik dari segi instalasi ROM/Software, perbaikan kerusakan pada smartphone serta informasi-informasi yang setiap user bisa saling share satu sama lain.

    Contoh yang ingin saya ambil inspirasinya adalah website xda developer. Di wesite itu semua komunitas smartphone di seluruh dunia berkumpul dan berbagi. Mungkin kelak di website yang saya inginkan kelak bisa menjembatani user di Indonesia yang masih belum paham tata cara tweak dan modding smarphone mereka.

  3. Bagaimana kira-kira kamu memproses pembuatan website dari awal hingga akhir?

    4. Pertama tentunya melihat-lihat website yang sudah ada dan besar komunitasnya seperti yang saya sampaikan diatas. Website xda developer mungkin akan menjadi rujukan awal saya.

    Setelah itu saya melakukan perencanaan seperti desain web seperti apa yang cocok diaplikasikan untuk web yang ingin saya buat. Setelah itu proses pengerjaan seperti koding dan lain-lain. Kemudian dilanjutkan dengan percobaan kepada user
